# Contributing

Thanks for your interest in contributing!

## Scope
- Primary script: **`mirror.sh`**. Describe behavior changes in the PR.
- Verify on a current **Ubuntu** (ideally 22.04 and 24.04) if you touch repository or APT logic.

## Steps
1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a Pull Request

All contributions will be reviewed.
